# 📘 CCNA 200-301 Course Notes

Welcome to my CCNA 200-301 study repository!

This repository contains my personal notes, Packet Tracer labs, and study materials while following Jeremy's IT Lab Complete CCNA course. Each day's learning is organized into separate folders with detailed explanations, diagrams, CLI commands, and lab exercises.

## 🎯 Goal

- Complete the CCNA 200-301 syllabus
- Build strong networking fundamentals
- Practice Packet Tracer labs daily
- Prepare for the Cisco CCNA certification exam
