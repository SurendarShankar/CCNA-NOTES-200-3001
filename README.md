# 📘 CCNA 200-301 Course Notes

Welcome to my CCNA 200-301 study repository!

This repository is inspired by an excellent CCNA notes repository that motivated me to organize my own study materials.

A special thanks to the original author for creating and sharing such a well-structured resource. I created this repository for my personal CCNA learning journey, where I can organize my notes, Packet Tracer labs, and study materials while following Jeremy's IT Lab Complete CCNA course.

## 🎯 Goal

- Complete the CCNA 200-301 syllabus
- Build strong networking fundamentals
- Practice Packet Tracer labs daily
- Prepare for the Cisco CCNA certification exam
